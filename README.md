#DiDa!準備上班–雲端差勤打卡系統APP
## 摘要
雲端差勤打卡系統為取代傳統打卡機，主要設計於行動裝置與網頁進行使用，自動化管理員工出缺勤狀況。

主要負責於APP之UI/UX設計、功能與程式設計與開發，並透過敏捷型開發，不斷根據客戶需求進行調整。

其APP介面透過AdobeXD設計原型後，程式開發透過Xamarin開發雙平台實現MVVM模式並串接後端與GoogleAPI等。
